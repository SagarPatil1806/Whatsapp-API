# WhatsApp Cloud API Integration (Node.js)

A Node.js implementation for interacting with the WhatsApp Business Platform (Cloud API). This project supports sending templates, text messages, media messages, and uploading media to Meta's servers.

## 🚀 Features
* **Template Messages:** Send pre-approved interactive or text templates.
* **Text Messages:** Send standard outbound text messages.
* **Media Messages:** Send images using either a hosted link or a Facebook Media ID.
* **Media Upload:** Upload local files to the WhatsApp Cloud storage to obtain a `media_id`.

## 🛠 Prerequisites
- Node.js installed.
- A Meta for Developers account with a configured WhatsApp Business App.
- A valid **System User Access Token**.
- A **Phone Number ID** from your WhatsApp dashboard.
